# DEEPDIABETIC - Expanded Django Project (Final Code)

This expanded project builds on the previous skeleton and adds:
- User authentication (login/logout) and simple user profile
- Admin + Django built-in auth
- Dashboard with dataset statistics and a Chart.js chart (static JS, no external fetch)
- CSV export of FundusImage metadata
- Simple alerts, improved responsive styling
- Management command `create_dummy_users` to create test users (no images or datasets included)
- All earlier features retained (core, datasets, diagnosis)

**No datasets or heavy model weights included.** See `diagnosis/model.py` for integration instructions.

Run locally (same steps as before).
